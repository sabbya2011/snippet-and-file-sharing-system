Snippet Bucket
A Web Application for Code Sharing

Use Snippet Bucket to store and share your snippets properly. The application is designed to take care your notes, snippets and source codes. Moreover you can join or create specific classroom to share and manage all your codes and notes in much better way. Snippet bucket provides you your personal wall for all the private codes you want to access in future. Use it once an get the essence of using a web application solely meant for your code management.
